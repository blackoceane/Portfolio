# Portfolio Web - HTML/CSS/JavaScript
Portfolio personnel développé avec **HTML**, **CSS** et **JavaScript**, présentant mes compétences de base en intégration web et en création d'interfaces simples et responsives.
##  Objectifs du projet
- Créer un portfolio **responsive** 
- Implémenter un **menu burger** animé avec JavaScript qui s' affiche dans le responsive
- Développer une **validation de formulaire** client-side
